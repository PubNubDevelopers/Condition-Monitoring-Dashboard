Predictive Maintenance Dashboard
Welcome to the Predictive Maintenance Dashboard Demo powered by PubNub. This demo shows real data from an industrial programmable logic controller connected to vibration sensors. This data is used to anticipate when equipment is about to fail or when maintenance is needed. Some features of this dashboard are disabled. Send an email to devrel@pubnub.com to learn more about how to create your own predictive maintenance monitoring system powered by PubNub.
